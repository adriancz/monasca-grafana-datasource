The json files in this directory are meant to be a starting point for useful
dashboards, but may require modification to work in various enviroments.  Some
values that may likely need changing:

1. Region (many have templated values of 'NCW' and 'NCE')
2. Tags (if tags are used, may need to change to fit your environment)
3. Data source (most assume monasca-grafana-datasource as datasource name)
